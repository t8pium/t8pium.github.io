Instagram Highlight Backup + Story Restore Toolkit

A local Python toolkit for personal highlight archival and restoration workflows, built around manual authentication, dedicated browser profiles, ordered folders, and calibrated screen-vision controls.

Positioning

This is framed as a responsible browser-assisted personal archive workflow for content the user owns or has permission to archive or publish.

Backup side

  • Launches or connects to a dedicated Chrome profile.
  • Uses manual login instead of collecting credentials.
  • Uses the currently opened Instagram profile page.
  • Detects highlight links and keeps the workflow local.

Restore side

  • Uses a folder picker to choose an ordered local highlight folder.
  • Calibrates visible buttons on the uploader extension page.
  • Steps through Add, Next, Publish now, and OK.
  • Waits 60-70 seconds after OK before the next file.
  • Supports dry runs, start position, and limits for resuming.

Boundary

The tool does not automate login, verification, CAPTCHA, account checks, or security decisions. It depends on the user controlling the browser and using the tool responsibly.
