# Usage boundaries

This project is intended for personal archival of content you own or have permission to save.

Do not use it to republish, mass-collect, or archive other people's content without permission. It is not affiliated with Instagram or Meta. Instagram's official data export remains the safest official route for retrieving your own account data.

The tool uses a local browser session and manual login. Keep browser profile folders, exported media, and local output directories out of version control.
