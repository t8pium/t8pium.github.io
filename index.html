Featured project / Built 2024

Computer-Vision Robotic Hand

A camera-based robotic hand system built with Python, computer vision, Arduino, and servo motors. The project turns live hand movement into motion on a physical prototype.

Camera
Arduino + Servos
Problem

I wanted to understand how software can control a physical system with vision input, closing the loop between camera, code, and hardware.

Build

Live webcam frames move through a Python vision pipeline. Hand landmarks become angles, then serial data, then servo motion.

What I learned

Debugging physical systems means isolating each layer: code, wiring, mechanics, timing, and the math that connects them.

Next version

Smoother mechanics, lower latency, gesture recognition, and a sturdier 3D-printed hand instead of a cardboard prototype.
