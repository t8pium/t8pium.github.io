Featured Project — Built 2024

Computer-Vision Robotic Hand

A camera-based robotic hand system built with Python, computer vision, Arduino, and servo motors. The system tracks live hand movement through a webcam and maps the motion to a physical servo-driven hand prototype.

Problem

I wanted to understand how software can control physical systems using vision input — closing the loop between camera, code, and hardware.

Build

A webcam feeds live frames into a Python script running a computer vision pipeline. Detected hand landmarks are mapped to angles, sent over serial to an Arduino, which drives servo motors attached to a cardboard mechanical hand.

Tools Used
Python OpenCV / MediaPipe Arduino (C++) Servo Motors Serial Communication Cardboard Prototyping
What I Learned

Hardware-software integration is harder than either alone. Debugging a physical system means the bug might be in the code, the wiring, the mechanics, or the mapping math. I learned to isolate each layer separately and prototype fast under constraints.

Next Improvements

Improve the mechanical structure for smoother movement, reduce serial latency, add gesture recognition for discrete commands, and eventually rebuild the hand in 3D-printed PLA for a more durable version.
